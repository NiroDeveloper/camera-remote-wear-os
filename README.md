# Camera Remove for Wear OS

This is a Wear OS app to trigger the native camera on your smartphone.
The app works by using the Bluetooth HID Device API from Android.

![Screenshot of the Main UI](./images/main-ui.png)

## Connection Problems

The app most likely has problems connecting to the phone.
I was able to trace this back the problem to the Android API, but could not fix it.
That's why I have removed the app from the Play Store.

I will therefore not develop the app further, everyone is free to analyze the problem further and make suggestions if necessary.
